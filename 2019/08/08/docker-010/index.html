仓库

第一章节的时候我们就讲过docker仓库的概念，我们提到仓库就是镜像存储的地方，有点类似于github这种平台。

为什么需要自己建立仓库？

  1. docker的默认仓库是在国外的，所以下载效率很慢。

  2. 可以存储一些隐私数据，不被泄露。

  3. 高度可定制。

Registry 镜像

Docker 官方提供了一个便捷创建仓库的能力，为此创建了一个镜像来实现自建仓库。目前版本V2.0。
