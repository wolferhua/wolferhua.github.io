Docker入门教程（四）Dockerfile（上）-自定义镜像

2019-05-09 docker docker, linux 0 评论 字数统计: 951(字) 阅读时长: 4(分)

之前我们使用的都是官方提供的一些镜像，这些镜像有一个共同点：大众可用。任何人都可以开箱使用它，但是实际工作中。我们总有这样那样的一些需求，这些官方提供的镜像就满足不了我们，我们就需要对镜像做一些个性化的操作。

为了方便学习，我们提一个虚拟的需求：对nginx进行自定义，打开首页显示'Hello Word'。
