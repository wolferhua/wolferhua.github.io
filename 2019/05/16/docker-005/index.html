ADD 和 COPY 区别

大家可能发现了，ADD 和 COPY 貌似实现了相同的功能，但是为啥要出现两个命令呢？是docker官方吃多了么？ – 尼古拉斯·码侠

ADD和COPY的区别：

  1. ADD如果添加的是一个压缩包 tar ，添加时会自动解压。
  2. ADD如果添加的是一个网址，那么会自动下载文件并添加。

ENTRYPOINT 和 CMD 区别

  1. 当我们运行容器时如果指定了命令，那么 CMD会被覆盖，但是 ENTRYPOINT 不会。
  2. 如果存在ENTRYPOINT ，CMD命令的值将作为ENTRYPOINT的参数。
